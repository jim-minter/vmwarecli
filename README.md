# vmwarecli

```
Usage: vmwarecli [options] cloneVMtoTemplate datacenter/vm/vm-name template-name
     | vmwarecli [options] deleteVM datacenter/vm/vm-name

Options:
  -h,--hostname <hostname>
  -k,--insecure
  -p,--password <password>
  -u,--username <username>

See https://github.com/jim-minter/vmwarecli for more details.
```
